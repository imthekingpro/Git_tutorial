프로그래밍 언어

프로그래밍 언어란 컴퓨터에서 일을 수행하는 프로그램을 만들기 위해 사용하는 기호체계를 의미한다.
코뮤니티의 모각코활동을 통해 다양한 프로그래밍 언어를 경험할 수 있다.

목차

  1. C언어
  2. 파이썬
  3. R언어

C언어

  • 많은 프로그램 개발에 사용되는 언어
  • 유닉스 운영체제에서 사용하기 위해 만들어진 언어
  • ->c언어 공부는 모각코에서!

파이썬

  • 비영리 재단이 운영하는 개방형, 공동체 기반 개발 모델을 지닌 언어
  • 개발에 막 입문한 이들이 사용하기 좋은 언어
  • ->python 공부도 모각코에서!

R언어

  • 데이터 분석에 주로 활용되는 언어
  • 데이터 분석에 관심이 많아서 넣어본 언어

    • R언어는.... 혼자 공부하는 걸로
